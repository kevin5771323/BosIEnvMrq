# 项目简介

## 核心功能
- 基于YOLOv8的水果蔬菜新鲜度检测系统，支持25类水果蔬菜的识别
- 支持多种输入源检测：单张图片、图片文件夹、视频文件、本地摄像头
- 提供用户友好的图形界面，可实时显示目标位置、目标总数和置信度
- 支持批量检测功能，可查看所有检测结果
- 提供模型选择功能和参数调整选项（置信度、IOU阈值）
- 支持检测结果保存和显示方式切换

## 适用场景
- 农产品质量检测与分级
- 超市和生鲜市场的水果蔬菜新鲜度监控
- 农业自动化生产管理系统
- 食品安全检测与研究
- 智能仓储管理系统

## 技术特点
- 基于深度学习的目标检测算法YOLOv8
- 使用PyQt5开发的图形用户界面
- 支持多种数据输入格式和实时处理
- 集成了完整的深度学习环境配置
- 提供详细的运行教程和使用指南
